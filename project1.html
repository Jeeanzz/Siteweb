Projet 1 : Pipeline CI/CD Sécurisé

Description

Création d’un pipeline CI/CD intégrant des outils de sécurité comme SAST (Static Application Security Testing) et DAST (Dynamic Application Security Testing), déployé sur AWS avec Docker et Kubernetes.

Défis rencontrés

Le principal défi était d'intégrer des étapes de sécurité automatisées dans un pipeline CI/CD déjà en place sans perturber les cycles de développement existants.

Solutions mises en œuvre

  • Intégration de SonarQube pour l'analyse statique de code et la détection des vulnérabilités.
  • Configuration d'OWASP ZAP pour l'analyse dynamique des applications déployées.
  • Utilisation de Docker et Kubernetes pour gérer les environnements de déploiement isolés et reproductibles.

Résultats

Le pipeline automatisé a permis de réduire les vulnérabilités de 40 % tout en accélérant le processus de déploiement de 25 %. Les développeurs ont également bénéficié d'un retour rapide sur les failles de sécurité.

Impact Organisationnel

L'adoption de ce pipeline CI/CD a transformé les pratiques de développement de l'organisation en introduisant une culture de sécurité dès les premières étapes (shift-left security). Les équipes ont gagné en autonomie grâce à des outils intégrés et à des formations ciblées.
